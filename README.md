# hello-koa


How start a new project?

- Compile
    docker-compose build
    
- Create a project with yeoman 
    docker-compose run web yo koa
     
- Start the project
    docker-compose up


How exec an existing project?
- Compile
    docker-compose build
    
- Download deps
    docker-compose run web npm install
     
- Start the project
    docker-compose up
